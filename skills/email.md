# 添加邮件功能

帮用户为项目集成邮件发送能力。

## 步骤

### 1. 了解需求

**问题 1：邮件用途**
> 你需要发什么邮件？
> - 注册验证邮件
> - 密码重置邮件
> - 通知邮件（订单确认、状态更新）
> - 营销邮件（推广、新闻通讯）
> 可以多选。

### 2. 选择邮件服务

> 推荐方案：
> - Resend（最简单，API 友好，免费额度 3000 封/月）
> - SendGrid（功能全面，免费 100 封/天）
> - Nodemailer + Gmail（开发测试用，不推荐生产）
> 不确定就用 Resend。

### 3. 生成邮件代码

- **邮件发送工具** — 封装发送函数，支持 HTML 和纯文本
- **邮件模板** — 根据用途生成 HTML 邮件模板（简洁专业的样式）
- **环境变量** — API Key 配置，添加到 .env.example
- **调用示例** — 在注册/下单等业务流程中调用发送

模板内容包含：
- 品牌 Logo 位置
- 正文区域
- 行动按钮（如"验证邮箱"、"查看订单"）
- 底部退订链接（营销邮件必须有）

### 4. 测试

发送一封测试邮件到用户邮箱，确认收到且格式正常。

> 邮件功能已配置。测试邮件发送成功。
> 记得在邮件服务商后台验证你的发送域名，避免进垃圾箱。
