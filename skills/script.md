# 写脚本

帮用户快速生成自动化脚本，解决重复性工作。

## 步骤

### 1. 了解需求

> 你想自动化什么操作？比如：
> - 批量重命名文件
> - 批量处理图片（压缩、转格式、调大小）
> - 从 CSV/JSON 中提取和转换数据
> - 定时备份文件
> - 批量替换文件内容
> - 抓取网页数据
> 直接描述你要做什么就行。

### 2. 选择语言

根据任务类型推荐：
- **Bash** — 文件操作、系统命令（最简单直接）
- **Python** — 数据处理、网络请求、复杂逻辑（最通用）
- **Node.js** — 涉及 JSON、API、Web 相关的（如果项目是 Node）

如果用户没有偏好，默认用 Python。

### 3. 生成脚本

生成的脚本必须包含：

- **使用说明** — 脚本开头的注释，说明用途和用法
- **输入验证** — 检查参数和文件是否存在
- **进度输出** — 处理多文件时显示进度
- **错误处理** — 单个文件出错不中断整个流程
- **dry-run 模式** — 加 --dry-run 参数可以预览操作而不执行

### 4. 测试

用少量样本数据测试脚本，确认输出正确。

> 脚本已生成。先用 --dry-run 预览操作，确认无误后再正式运行。
